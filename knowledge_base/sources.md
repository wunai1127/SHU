## 已获取来源

### 1. Prot_000 OCS Protocol ✅

| 项目 | 信息 |
|------|------|
| **状态** | 已获取 |
| **内容** | OCS (Organ Care System) 操作参数 |
| **覆盖指标** | AOP, CF |
| **阈值可信度** | 高 |

**提供的参数**:
- 主动脉压操作范围: 40-100 mmHg
- 冠脉流量目标: 400-900 mL/min
- 警告线和操作建议

| **覆盖指标** | EF, CI, MAP, SvO2, Lactate, HR |

| 项目 | 信息 |
|------|------|
| **状态** | 仅供参考 |
| **内容** | 心血管系统模拟模型默认参数 |

## 来源可信度评级

| 来源类型 | 可信度 | 适用场景 |
|----------|--------|----------|
| 国际指南 (ISHLT, AHA/ACC) | 最高 | 直接用于临床决策 |
| 大型 RCT (PROCEED II) | 高 | 直接参考 |
| FDA 审批文件 | 高 | 设备相关参数 |
| 设备厂商协议 (OCS) | 高 | 特定设备操作 |
| 临床共识 | 中-高 | 一般性参考 |
| 专家意见 | 中 | 缺乏其他来源时参考 |
| 模拟模型 (RCVSIM) | 低 | 仅供参考，需验证 |
| 数据驱动推断 | 低 | 标记后可用，需定期复核 |

## 文献获取优先级

### 高优先级 🔴

1. **ISHLT 2022 心脏移植指南**
   - 原因: 权威性最高，覆盖面广
   - 建议: 通过医学图书馆或机构订阅获取

2. **PROCEED II 研究报告**
   - 原因: OCS 相关临床证据
   - 建议: 通过 PubMed/期刊获取

### 中优先级 🟡

3. **FDA SSED for OCS Heart**
   - 原因: 补充设备参数信息
   - 建议: FDA 官网检索

4. **ESC/AHA 心力衰竭指南**
   - 原因: 补充血流动力学管理
   - 建议: 学会官网下载

### 低优先级 🟢

5. **RCVSIM 模型文档**
   - 原因: 补充模型参数解释
   - 建议: 学术资源检索

## 验证和更新机制

### 定期复核

- **频率**: 每6个月
- **内容**:
  - 检查是否有新指南发布
  - 验证现有阈值是否仍为最佳实践
  - 更新置信度标记

### 新增来源流程

1. 获取文献
2. 提取相关阈值
3. 与现有配置对比
4. 更新配置文件
5. 记录来源和更新日期
6. 通知相关人员

### 冲突处理

当不同来源的阈值存在冲突时:
1. 优先采用可信度更高的来源
2. 记录冲突和决策理由
3. 标记需要临床专家复核
